基本语法

一般格式化

  • 一般参数依次填充
  • 可以在 {} 内写清楚标号（从 0 开始）然后 format() 内参数按序号填入 eg. print("{1} {0} {1}".format('hello', 'world'))
  • 也可以在 {} 内写命名，然后 format() 里带上 eg. print("name is {a}, from {b}".format(a="Alice", b="CN"))
  • ==两者可以混合使用==

数字格式化

  1. 整数格式化：
    1. 十进制：{:d}，正常表示
    2. 二进制：{:b}，使用符号 0b 或 0B 前缀
    3. 八进制：{:o}，使用符号 0o 或 0O 前缀
    4. 十六进制：{:x}（大写 X 也可），使用 0x 或 0X 前缀
  2. 浮点数格式化：
    1. 定点表示法：{:.2f} 或者 {:.2F}，其中 2 是保留几位小数，==遇到 inf（无穷大）和 nan（非数字）时候，会直接输出 inf / nan，若是大写 F 则为 INF 和 NAN==
    2. 科学计数法：{:.2e} 或者 {:.2E}，数字意思同上，大小写决定输出 E 大小写
    3. 百分表示法：{:.2%}，输出百分之形式和 %
    4. 通用表示法：{:.2g} 或者 {:.2G}，
