1. 注释

  • 单行注释：#
  • 多行注释：”””  ””” 或者’’’  ’’’这样括起来
  • tips：有变量接收叫多行字符串｜没有变量接收叫多行注释

2. 变量

  • 格式 ：x=……
  • **字符串变量还是两种引号都行 **

[!notes]- Features :
1. 能一行声明多个变量 ：x, y, z = “banana”, “apple, “pear”
2. 能多个变量赋同值：x = y = z = “banana”

  • print () 可以打印文本 & 变量

和

  • “+” 可以拼接任意相等的成分（包括本身和对应的变量）

  • ==如果所拼接成分不相等，可以用强制类型转换== == Eg. str(918230) + “is my number” ==

  • 全局变量

  1. 可以在所有函数外声明
  2. 在函数内创建或引用时，加 global 字样
  1. 函数内引用必须加 global 啥啥啥，在global前面的同名变量在前面运作不受影响，在其后被替换
  2. 如果前面没有，但是使用了，Python 会按作用域逐层向外搜索该名变量

4. 数字

  • 数字类型：int, float, complex
  • ==float 型有两种写法：89.23435 & -49.8e100/15E2== E/e 不分大小写，且 e 前面可以是大于 10 的数
  • 随机数：有模块

[!notes]

  • complex 型用 j 做虚数单位
  • 强制类型转换中，float 变 int ==直接截断小数== int 变float ==会加一个 0== int 变 complex ==会有 0j==
  • 上面说的是打印出的值
  • 进制转换：十进制转换其他：bin(), oct(), hex() 对应 binary、octal、hexadecimal 其他转十进制：int(number, scale) 前面填数字，后面填进制（2、8、16）
  • 进制表示：十进制为 d，直接写没符号 二进制为 b，前缀 0b 或 0B 八进制为 o，前缀 0o 或 0O 十六进制为 x 或X，前缀 0x 或 0X

6. 字符串

  • "" 或者 '' 均可，两者相等
  • 多行字符串：""" """ 或者 ''' ''' 赋给变量就行（不赋给就是多行注释）
  • ==字符串也是数组==

[!note] 包括单个字符，它就是==长度为 1 的字符串== 当有：a = "hello!" 时，就相当于创建了一个长度为 6 的数组 因此你可以用 print(a[1]) 打印出 e

  • 对字符数组的功能
    • 裁切

[!question] Q：能 b[-1:-3] 吗？ A：可以，裁剪长度负值，剪下内容为空（输出一行空）

Q：能 b[1:-3] 吗？ A：可以，输出 ello, Wor（中间内容）

Q：能 b[:-3] 吗？ A：可以，输出 Hello, Wor（从 0 开始）

Q：能 b[1:] 吗？ A：==可以，输出 ello, World!==(注意这个，没有左闭右开的问题)

Q：能 b [-99:-3] 吗？（就是能超出吗） A：可以，会显示所有 有东西的位置

  • 索引

    ==所以索引不能超出范围，切片可以==

    • 拼接（级联/串联）
      1. 变量和变量 / 字符间用“+”
      2. 字符串和字符串之间==可以==直接拼（加空格不影响，显示没有）
      3. 要在字符串间加空格，可以 a + " " + "hello" 这样
      4. 不同形式变量不能直拼接，可以强制类型转换，也可以用 format() 方法
